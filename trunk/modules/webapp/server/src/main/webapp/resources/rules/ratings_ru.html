WiseMatches использует Систему Рейтингов Эло. Рейтинг игра пересчитывается после завершения очередной игры.

  1. Начальный Рейтинг

    1. Про создании нового акаунтра игрок получает рейтинг 1200.

  2. Изменение Рейтинга

    1. Если игра участвует в рейтинге, рейтинг всех игроков изменяется после ее завершения.

    2. Если игрок играет с более чем одним противником, его/ее рейтинг рассчитывается по правилам мини турнира, как если бы он/она сыграла две или больше игр. Хотя игры с несколькими игроками имеют только одного победителя, игрок может выиграть одного и проиграть другому противника и его рейтинг будет изменен соответствующим образом.

  3. Значение Коэффициента K

    1. Уравнивающий коэффициент принимает значение 16 для профессионалов (чей рейтинг 2200 и выше) и значение 32 для остальных игроков.

  4. Игры, не участвующие в рейтинге

    1. Игра не участвует в рейтинге если один из игроков сделал меньше трех ходов до завершения игры

    2. Все игры, в которых участвует игрок Гость не участвуют в рейтингах.

  5. Рейтинги Роботов

    1. Все роботы имеют предопределенные рейтинги:

      1. 800 для Робота Труса;

      2. 1200 для Робота Балбеса;

      3. 1800 для Робота Бывалого.

    2. Рейтинги роботов никогда не изменяются.
